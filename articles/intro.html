Dual Grid Voxels

Dual Grid Voxels is a mixed-initiative level editor that decorates a designer-authored voxel grid with artist-authored tiles. This magnifies artist effort and makes it easier for level designers to create good looking levels quickly.

It imports 3D cube-tiles from an artist, automatically extracting mark-up and preparing the tiles for use with the voxel grid. The importer has a simple interface and can import a simple set of tiles without you having to configure it. After initial set up, artists can reimport their models with a single click in Unity.

As your tilesets get larger and you get more experience with the tool, you can configure the importer or edit the source code to have full control on how the models are imported. This will not affect the artist's workflow - they still press a single button to reimport their models.

When to use Dual Grid Voxels

It is strongest when iteratively developing and changing large worlds with a consistent style. It can also be used as a 'frontend' for voxel procedural generation. The algorithm can be run asynchronously, supporting its use in level loading.

It provides less benefit if you are working on distinct scenes without many shared elements. Running the placer algorithm on larger voxel grids and complicated tilesets can take 100s of milliseconds, and some voxel changes could lead to decoration changes far away from where the change was made. Using it in a Minecraft-like would require care and dilligence.
