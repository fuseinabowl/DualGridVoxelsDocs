Class GridPlacer

This Unity component manages the grid placing during edit time and runtime.

It will automatically generate a mesh when the world starts, both in editor and in the game. It can also generate the mesh later by calling GenerateNow(Boolean, Func<GameObject, Transform, GameObject>).
