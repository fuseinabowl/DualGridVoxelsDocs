Class VoxelMesher

This class manages a naive mesh for a voxel grid.

It can generate the naive mesh and can map any face index on the generated mesh to a voxel face. This is used by to select which voxel to fill/clear when clicking on the voxel grid.

Gets the VoxelMesher.FaceData for a triangle in the generated mesh. The mesh must have been generated first by calling RegenerateMeshedVoxels(VoxelGrid, Vector3).

The triangle index is expected to come from a physics raycast based on where the user clicks.
